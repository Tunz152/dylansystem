Privacy Policy for Dylan System

Effective Date: May 22, 2025

Dylan System is a Discord bot developed to assist with moderation, logging, and interactive features. This privacy policy explains how we handle user data.

What data do we collect?

Dylan System may temporarily access the following data:

  • Message content (for moderation, suggestions, or auto-reply features)
  • User IDs (to tag users or log commands)
  • Server and channel IDs (to perform actions in the correct location)

Do we store your data?

No. Dylan System does not permanently store message content or user data on any external server. All data is processed in-memory and used only to execute commands.

Who can access the data?

Only the bot owner and authorized developers have access to runtime logs (if any). We do not share any data with third parties.

How can users opt out?

Users can remove the bot from their server at any time. Bot administrators may also disable features that rely on message content.
